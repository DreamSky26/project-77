WHAT IS VISUAL STUDIO CODE?


Visual Studio is an integrated development environment from Microsoft. It is used to develop computer programs, as well as websites, web apps, web services and mobile apps.



WHAT IS CONSOLE SCREEN


Console is the infrastructure for console applications in Microsoft Windows. An instance of a Windows Console has a screen buffer and an input buffer.

If you want to go to the console screen the just do the following:

1. Right click on your mouse.

2. You will be able to see some options, click on "Inspect".

3. When you see something appearing on the right side of your computer, that is the inspect area. You might see a section at the top of the inspect area where there are things like "Sources", "Elements" and all, click on "Console".

4. And you are in the console screen. Scroll-up in the inspect area console screen to see.
